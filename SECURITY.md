# Security Policy

## Supported Versions

Indom is currently in active MVP development.

| Version | Supported |
| ------- | --------- |
| latest (main) | ✅ |
| older commits | ❌ |

## Important Note

Indom runs **100% locally** on your machine.
- No data is sent to any server we control
- Your CV never leaves your machine
- API keys are stored locally only
- We have no access to your data at any point

## Reporting a Vulnerability

If you find a security vulnerability in Indom:

1. **Do NOT open a public GitHub issue**
2. Email us directly at: yugdabgar21@gmail.com
3. Include:
   - Description of the vulnerability
   - Steps to reproduce
   - Potential impact
   - Suggested fix (if any)

We will respond within **48 hours** and aim to fix
critical issues within **7 days**.

Thank you for helping keep Indom safe. 🔒
